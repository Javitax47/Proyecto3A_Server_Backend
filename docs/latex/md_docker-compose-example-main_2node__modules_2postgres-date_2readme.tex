\begin{quote}
Postgres date output parser \end{quote}
This package parses \href{https://www.postgresql.org/docs/current/datatype-datetime.html\#DATATYPE-DATETIME-OUTPUT}{\texttt{ date/time outputs}} from Postgres into Javascript {\ttfamily Date} objects. Its goal is to match Postgres behavior and preserve data accuracy.

If you find a case where a valid Postgres output results in incorrect parsing (including loss of precision), please \href{https://github.com/bendrucker/postgres-date/compare}{\texttt{ create a pull request}} and provide a failing test.

{\bfseries{Supported Postgres Versions\+:}} {\ttfamily \texorpdfstring{$>$}{>}= 9.\+6}
