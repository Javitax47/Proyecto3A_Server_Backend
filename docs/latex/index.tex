\begin{DoxyItemize}
\item GET /setup\+: Crea las tablas de usuarios y sensores en la base de datos.
\item GET /latest\+: Obtiene las últimas mediciones de temperatura y CO2.
\item GET /\+: Devuelve todos los usuarios y sensores.
\item POST /\+: Inserta un nuevo sensor.
\begin{DoxyItemize}
\item Body\+: \{ "{}type"{}\+: "{}temperature"{}, "{}value"{}\+: 25.\+5, "{}timestamp"{}\+: "{}2024-\/09-\/22\+T12\+:00\+:00\+Z"{}, "{}user\+Id"{}\+: 1 \}
\end{DoxyItemize}
\item POST /users\+: Crea un nuevo usuario.
\begin{DoxyItemize}
\item Body\+: \{ "{}username"{}\+: "{}nuevo\+\_\+usuario"{} \}
\end{DoxyItemize}
\item DELETE /users/\+:id/measurements\+: Elimina todas las mediciones de un usuario específico.
\item DELETE /reset\+: Reinicia las tablas de la base de datos.
\item DELETE /erase\+: Elimina todas las tablas.
