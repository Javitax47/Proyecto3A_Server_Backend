\doxysubsection{Descripción detallada}
API para gestionar usuarios y sensores, incluyendo funcionalidades de creación, consulta, inserción y eliminación de datos. 

Este servidor Express gestiona las tablas de usuarios y sensores en una base de datos Postgre\+SQL. Proporciona rutas para interactuar con las entidades de usuarios, sensores, mediciones, alertas y resetear las tablas.
